# README

## MyAdmin
* rails version 7.0.0
* Bundler version 2.1.4
* npm version 6.14.15
* yarn version 1.22.17


### Download Application
```
git clone https://github.com/walterpaulo/myadmin.git
cd myadmin
bundle install
rails db:migrate
rails db:seed
yarn
```

### Start

```
rails server
```
**http://localhost:3000**
